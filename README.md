# React Hooks Slider

A simple slider component built with React Hooks.

## Usage

```bash
# clone the repo
git clone https://github.com/MKAbuMattar/react-hooks-slider.git

# install dependencies
cd react-hooks-slider
npm install

# run the app
npm run dev
```
